## 4.28 — Outro problema do else oscilante

Modifique o código dado para produzir a saída mostrada em cada parte do problema. Utilize técnicas de recuo adequadas. Não faça nenhuma alteração além de inserir chaves e alterar o recuo do código. O compilador ignora o recuo em um programa Java.

Código fornecido:

```java
if (y == 8)
if (x == 5)
System.out.println("@@@@@");
else
System.out.println("#####");
System.out.println("$$$$$");
System.out.println("&&&&&");
```

### a)

Supondo que `x = 5` e `y = 8`, a seguinte saída é produzida:

```text
@@@@@
$$$$$
&&&&&
```

### b)

Supondo que `x = 5` e `y = 8`, a seguinte saída é produzida:

```text
@@@@@
```

### c)

Supondo que `x = 5` e `y = 8`, a seguinte saída é produzida:

```text
@@@@@
```

### d)

Supondo que `x = 5` e `y = 7`, a seguinte saída é produzida. Todas as três últimas instruções de saída depois do `else` são partes de um bloco.

```text
#####
$$$$$
&&&&&
```
