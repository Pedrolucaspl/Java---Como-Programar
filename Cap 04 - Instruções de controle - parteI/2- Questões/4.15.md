## 4.15 — Identifique e corrija os erros

Identifique e corrija os erros em cada um dos seguintes fragmentos de código. Pode haver mais de um erro em cada trecho.

### a)

```java
if (age >= 65);
   System.out.println("Age is greater than or equal to 65");
else
   System.out.println("Age is less than 65)");
```

```java
if (age >= 65)
   System.out.println("Age is greater than or equal to 65");
else
   System.out.println("Age is less than 65");
```

### b)

```java
int x = 1, total=0;
while (x <= 10)
{
   total += x;
   ++x;
}
```
```java
int x = 1, total=0;
while (x <= 10)
{
   total += x;
   ++x;
}
```
### c)

```java
while (x <= 100)
   total += x;
   ++x;
```
```java
while (x <= 100){
   total += x;
   ++x;
}
```

### d)

```java
while (y > 0)
{
   System.out.println(y);
   ++y;
```
```java
while (y > 0)
{
   System.out.println(y);
   --y;
}
```
