# 3.9 — Usando uma classe sem importá-la

(Usando uma classe sem importá-la) Explique como um programa pode usar a classe `Scanner` sem importá-la.

**Resposta:** Um programa pode usar a classe `Scanner` sem importá-la utilizando seu nome totalmente qualificado, isto é, incluindo o nome do pacote ao qual ela pertence. Como a classe `Scanner` está no pacote `java.util`, ela pode ser utilizada da seguinte forma:

```java
java.util.Scanner entrada = new java.util.Scanner(System.in);
```
