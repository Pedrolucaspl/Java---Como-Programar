# 3.6 — Construtores padrão

(Construtores padrão) O que é um construtor padrão? Como as variáveis de instância de um objeto são inicializadas se uma classe tiver somente um construtor padrão?

**Resposta:** Construtor padrão é um construtor sem parâmetros que o Java fornece automaticamente quando uma classe não tem nenhum construtor. Quando uma classe possui apenas um construtor padrão, as variáveis de instância são inicializadas com valores padrão definidos pelo Java, como `0` para tipos numéricos, `false` para `boolean` e `null` para tipos de referência.
