# 2.13 — Ordem de avaliação e valor final de x

Declare a ordem de avaliação dos operadores em cada instrução e indique o valor de `x` após executá-la.

## a)

```java
x = 7 + 3 * 6 / 2 - 1;
```

### Ordem de avaliação

1. `3 * 6 = 18`
2. `18 / 2 = 9`
3. `7 + 9 = 16`
4. `16 - 1 = 15`

### Valor final de x

```java
x = 15;
```

---

## b)

```java
x = 2 % 2 + 2 * 2 - 2 / 2;
```

### Ordem de avaliação

1. `2 % 2 = 0`
2. `2 * 2 = 4`
3. `2 / 2 = 1`
4. `0 + 4 = 4`
5. `4 - 1 = 3`

### Valor final de x

```java
x = 3;
```

---

## c)

```java
x = (3 * 9 * (3 + (9 * 3 / (3))));
```

### Ordem de avaliação

1. `9 * 3 = 27`
2. `27 / 3 = 9`
3. `3 + 9 = 12`
4. `3 * 9 = 27`
5. `27 * 12 = 324`

### Valor final de x

```java
x = 324;
```
