# 2.11 — Variáveis com valores modificados



Quais instruções abaixo contêm variáveis cujos valores são alterados?

a)

```java
p = i + j + k + 7;
```

b)

```java
System.out.println("variables whose values are modified");
```

c)

```java
System.out.println("a = 5");
```

d)

```java
value = input.nextInt();
```

---

## As instruções que contêm variáveis cujos valores são modificados são:

- a) `p = i + j + k + 7;`
- d) `value = input.nextInt();`

As instruções b) e c) apenas exibem mensagens na tela.
