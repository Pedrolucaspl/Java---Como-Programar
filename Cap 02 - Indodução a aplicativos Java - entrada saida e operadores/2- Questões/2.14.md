# Exercício 2.14

Escreva um aplicativo que exiba os números 1 a 4 na mesma linha, com cada par de adjacentes separados por um espaço.

## a) Uma instrução `System.out.println`

```java
public class Exercicio214A {
    public static void main(String[] args) {
        System.out.println("1 2 3 4");
    }
}
```

### Saída

```text
1 2 3 4
```

---

## b) Quatro instruções `System.out.print`

```java
public class Exercicio214B {
    public static void main(String[] args) {
        System.out.print("1 ");
        System.out.print("2 ");
        System.out.print("3 ");
        System.out.print("4");
    }
}
```

### Saída

```text
1 2 3 4
```

---

## c) Uma instrução `System.out.printf`

```java
public class Exercicio214C {
    public static void main(String[] args) {
        System.out.printf("%d %d %d %d%n", 1, 2, 3, 4);
    }
}
```

### Saída

```text
1 2 3 4
```
