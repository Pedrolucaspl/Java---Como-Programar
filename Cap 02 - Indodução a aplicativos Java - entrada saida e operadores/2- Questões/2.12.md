# 2.12 — Equação y = ax³ + 7


Dado $y = a x^3 + 7$, quais alternativas abaixo representam corretamente essa equação em Java?

a)

```java
y = a * x * x * x + 7;
```

b)

```java
y = a * x * x * (x + 7);
```

c)

```java
y = (a * x) * x * (x + 7);
```

d)

```java
y = (a * x) * x * x + 7;
```

e)

```java
y = a * (x * x * x) + 7;
```

f)

```java
y = a * x * (x * x + 7);
```
