# 2.29 — Valor inteiro de um caractere

Exiba os equivalentes inteiros dos caracteres:

```text
A B C a b c 0 1 2 $ * + / espaço
```

Exemplo:

```java
System.out.printf("The character %c has the value %d%n", 'A', ((int) 'A'));
