# 2.9 — Verdadeiro ou falso

a) Operadores em Java são avaliados da esquerda para a direita.

b) Os seguintes nomes são todos identificadores válidos de variável?
_under_bar_, m928134, t5, j7, her_sales$, his_$account_total, a, b$, c, z e z2


c) Uma expressão aritmética Java válida sem parênteses é avaliada da esquerda para a direita.

d) Os seguintes nomes são todos identificadores inválidos de variável?
3g, 87, 67h2, h22 e 2h

---

a) Falso — os operadores são avaliados de acordo com a precedência dos operadores e associatividade, não apenas da esquerda para a direita.

b) Verdadeiro — todos os nomes apresentados seguem as regras de identificação de variáveis em Java.

c) Falso — expressões aritméticas sem parênteses seguem a precedência dos operadores antes da avaliação da esquerda para a direita.

d) Falso — `h22` é um nome de variável válido, pois começa com uma letra.
