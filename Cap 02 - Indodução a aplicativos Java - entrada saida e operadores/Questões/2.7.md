# 2.7 — Preencha as lacunas


a) `Cometários` são usados para documentar um programa e melhorar sua legibilidade.

b) Uma decisão pode ser tomada em Java com um(a) `intrução if`.

c) Cálculos normalmente são feitos por instruções `atribuição`.

d) Operadores aritméticos com a mesma precedência de multiplicação: `/` e `%`.

e) Quando parênteses estão aninhados, o conjunto `interno` é avaliado primeiro.

f) Uma posição na memória cujo valor pode mudar durante a execução do programa é uma `variável`.
