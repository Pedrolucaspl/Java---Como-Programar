# 2.10 — O que cada instrução exibe? (x = 2, y = 3)


a)

```java
System.out.printf("x = %d%n", x);
```
x = 2
b)

```java
System.out.printf("Value of %d + %d is %d%n", x, x, (x + x));
```
Value of 2 + 2 is 4
c)

```java
System.out.printf("x =");
```
x = 
d)

```java
System.out.printf("%d = %d%n", (x + y), (y + x));
```
5 = 5
