## 2.4 — Identifique e corrija os erros

### a)

```java
if (c < 7);

   System.out.println("c is less than 7");
```
```java
if(c<7)
    System.out.println("c is less than 7");
```

### b)

```java
if (c => 7)

   System.out.println("c is equal to or greater than 7");
```
```java
if (c >= 7)

   System.out.println("c is equal to or greater than 7");
```
