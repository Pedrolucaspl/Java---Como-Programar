# Exercício 1.12 — Neutralidade de gêneros

Explique como você desenvolveria um algoritmo capaz de substituir termos específicos de gênero por equivalentes neutros em um texto.

Exemplos:

- “wife” / “husband” → “spouse”
- “man” / “woman” → “person”
- “daughter” / “son” → “child”

Considere também possíveis problemas gerados pelas substituições automáticas.
