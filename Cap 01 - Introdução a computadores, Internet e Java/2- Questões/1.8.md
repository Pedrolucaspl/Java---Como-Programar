# Exercício 1.8

Um dos objetos mais comuns do mundo é um relógio de pulso. Discuta como cada um dos seguintes termos e conceitos se aplicam à noção de um relógio:

- objeto: relógio de pulso
- atributos: cor, modelo, ano
- comportamentos: mostar hora, ajustar horário
- classe: relogio
- herança: smartWatch
- modelagem
- mensagens: pressione o botão
- encapsulamento: o horário não deve ser alterado diretamente
- interface: botões
- ocultamento de informações: como os circuitos funcionam
