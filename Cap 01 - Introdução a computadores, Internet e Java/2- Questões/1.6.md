# Exercício 1.6

Preencha as lacunas em cada uma das seguintes afirmações.

a) Os programas Java normalmente passam por cinco fases: **Criar**, **Compilar**, **Carregar**, **Verificar bytecode** e **Executar**.

b) Um(a) **IDEs** fornece muitas ferramentas que suportam o processo de desenvolvimento de software, como editores para escrever e editar programas, depuradores a fim de localizar erros de lógica em programas e muitos outros recursos.

c) O comando `java` invoca **JVM**, que executa programas Java.

d) Um(a) **Máquina virtual** é um aplicativo de software que simula um computador, mas oculta o sistema operacional e o hardware subjacentes dos programas que interagem com ela(e).

e) O **carregador de classes** transfere os arquivos `.class` contendo os bytecodes do programa para a memória principal.

f) O **verificador de bytecodes** examina bytecodes para assegurar que eles são válidos.
